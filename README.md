# PrepWise AI 🎯

PrepWise AI is a full-stack web application built using the MERN stack that helps users prepare for interviews in a smarter way. It uses AI to generate interview questions and analyze responses, making practice more realistic and effective.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 🤖 AI-powered interview question generation
- 🧠 Real-time interview simulation
- 📊 Interview report & performance analysis
- 🛡️ Protected routes for secure access
- 🌐 Clean and responsive UI

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Vite
- SCSS

**Backend**
- Node.js
- Express.js
- MongoDB

**Other**
- JWT Authentication
- REST APIs
- AI integration (for interview processing)
